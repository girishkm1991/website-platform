# Website Platform

A scalable, Docker-based website generation platform built with React, Express.js, MySQL and Nginx.

## Vision

Build a modern platform capable of generating professional websites for multiple industries using reusable templates, themes, plugins and dynamic content.

## Technology Stack

- React
- Express.js
- MySQL
- Docker
- Docker Compose
- Nginx
- GitHub
- AWS EC2
